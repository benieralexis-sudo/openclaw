# Proactive Agent

Agent proactif de MoltBot. Rapports automatiques, alertes pipeline, detection de hot leads.

## Fonctionnalites automatiques
- Rapport matinal (8h) : resume activite overnight
- Alertes pipeline (9h) : deals stagnants, dates de cloture proches
- Rapport hebdomadaire (lundi 9h) : bilan complet
- Rapport mensuel (1er du mois 9h) : bilan + tendances
- Check emails (toutes les 30 min) : detection hot leads (3+ opens)
- Analyse nocturne (2h) : preparation briefing matinal

## Commandes
- "active/desactive le mode proactif" — on/off
- "configure les alertes" — modifier horaires et seuils
- "rapport maintenant" — rapport quotidien immediat
- "rapport de la semaine" — rapport hebdo immediat
- "rapport du mois" — rapport mensuel immediat
- "mes alertes" — voir la config
- "historique des alertes" — derniers envois
- "mode proactif status" — statut detaille
