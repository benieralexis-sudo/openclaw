## 4. ANALYSE STRATEGIQUE — QUESTIONS CLES

### Frais de setup
| Type | Setup fees |
|------|-----------|
| Outils SaaS (Instantly, Smartlead, etc.) | **$0** — jamais |
| Agences mid-market (Cleverly, Sopro) | **$0** en general |
| Agences premium (CIENCE) | **$5 000** (sprint 5j) |
| Agences enterprise (SalesRoads) | Inclus dans le retainer |
| AI SDR (AiSDR, Salesforge) | **$0** |

**Conclusion** : les top performers ne facturent PAS de setup, sauf les plus chers (CIENCE). Le setup gratuit est devenu la norme car il reduit la friction d'acquisition.

### Duree d'engagement standard
| Modele | Frequence |
|--------|-----------|
| Sans engagement (mensuel) | **Le plus courant** pour SaaS et AI SDR |
| Pilote 3 mois | Courant pour agences (Martal, mid-market) |
| Pilote 4 mois | Premium (Martal Tier 2-3) |
| 6 mois minimum | Rare, surtout pour custom/enterprise |
| 12 mois | Rare, surtout pour remises importantes |
| Trimestriel | Standard AI SDR (AiSDR, Salesforge Agent Frank) |

**Tendance 2025-2026** : le marche evolue vers MOINS d'engagement. Les agences qui gagnent proposent des pilotes 3 mois puis mensuel. Les outils AI SDR sont sans engagement.

### Volume discounts
- **Annuel vs mensuel** : quasi-universel, 10-20% de remise
- **Volume leads** : oui pour les outils (Instantly, Saleshandy = tiers progressifs)
- **Multi-seats** : Waalaxy, Lemlist = prix/user qui baisse en Enterprise
- **Agences** : remises implicites sur les tiers superieurs, pas de grille publique

### Pay-per-meeting / Pay-per-result
| Agence | Pay-per-meeting ? |
|--------|-------------------|
| Acceor | **NON** — le rejette explicitement |
| Belkins | NON — retainer fixe |
| CIENCE | HYBRID — commission en option sur meetings |
| Cleverly | NON — retainer fixe |
| SalesRoads | NON — retainer fixe |
| Martal Tier 2-3 | HYBRID — fee fixe + commission sur ventes |
| AiSDR | INDIRECT — estime ~3 ou ~11 meetings/mois |

**Conclusion** : le pay-per-meeting pur est RARE chez les bonnes agences. Les meilleures evitent car ca pousse a la quantite. Le modele dominant = **retainer fixe mensuel** avec objectifs indicatifs.

## 5. PSYCHOLOGIE PRICING — STRUCTURE OPTIMALE

### Nombre de tiers
| Structure | Qui l'utilise | Efficacite |
|-----------|---------------|------------|
| **3 tiers** | Instantly, Saleshandy, Lemlist, Martal, Belkins, Woodpecker, AiSDR | **LE PLUS COURANT ET EFFICACE** |
| **4 tiers** | Saleshandy (4e = Scale Plus), Woodpecker (4 = Max) | Courant pour SaaS avec large gamme |
| **2 tiers** | Salesforge (Pro + Growth) | Simple, SaaS low-ticket |
| **1 offre** | Cleverly (1 prix par canal), SalesRoads | Clair, pas de confusion |

### Structure qui convertit le mieux : **3 TIERS avec ancrage**

La structure dominante chez les leaders :
1. **Tier 1 "Starter/Growth"** : prix d'entree accessible, volume limite
2. **Tier 2 "Pro/Popular"** (marque "MOST POPULAR" ou "BEST VALUE") : 2-3x le prix, 10x le volume
3. **Tier 3 "Enterprise/Custom"** : devis, legitime le Tier 2 comme "bon deal"

**Techniques psychologiques observees :**
- **Ancrage** : le Tier 3 rend le Tier 2 raisonnable
- **Badge "Most Popular"** : utilise par Instantly, Saleshandy, Lemlist, AiSDR, Belkins
- **Prix annuel barre** : -10 a -20%, cree urgence
- **Essai gratuit** : 7-14 jours, quasi-universel pour SaaS
- **Prix en .90/.60** : Instantly ($37.60), pas de .99 (B2B)
